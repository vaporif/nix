# LibreWolf Reproducible Configuration Design

## Goal

Make LibreWolf installation reproducible on a fresh machine: auto-install 14 extensions via `policies.json` and set all preferences declaratively via `librewolf.overrides.cfg`. Both files generated by a new Nix module.

## Decisions

- **Extensions**: Auto-installed via `policies.json` `ExtensionSettings` with `installation_mode = "normal_installed"` (user can uninstall from UI)
- **Wallets excluded**: MetaMask and polkadot{.js} are NOT auto-installed (security-sensitive, manual install)
- **Preferences**: All use `defaultPref()` (overridable in about:config, not locked)
- **Search engines**: DuckDuckGo default, remove Google/Bing/Amazon/eBay/Twitter/Perplexity, add DuckDuckGo Lite/Searx/MetaGer/Startpage/Mojeek (preserved from LibreWolf's bundled policies)
- **Profile cleanup**: Not in scope (stale profiles are harmless)

## Architecture

### New file: `home/common/librewolf.nix`

Nix module that generates two files via `home.file`:

1. **`~/.librewolf/policies.json`** — extensions + search engines + existing LibreWolf policies
2. **`~/.librewolf/librewolf.overrides.cfg`** — all preferences (replaces static config file)

### policies.json structure

Merges LibreWolf's existing bundled policies with our extensions:

```json
{
  "policies": {
    "DisableAppUpdate": true,
    "DisableSystemAddonUpdate": true,
    "DisableFirefoxStudies": true,
    "DisableTelemetry": true,
    "DisableFeedbackCommands": true,
    "DisablePocket": true,
    "NoDefaultBookmarks": true,
    "OverrideFirstRunPage": "",
    "OverridePostUpdatePage": "",
    "SkipTermsOfUse": true,
    "ExtensionSettings": {
      "<extension-id>": {
        "install_url": "https://addons.mozilla.org/firefox/downloads/latest/<slug>/latest.xpi",
        "installation_mode": "normal_installed"
      }
    },
    "SearchEngines": {
      "Default": "DuckDuckGo",
      "Remove": ["Google", "Bing", "Amazon.com", "eBay", "Twitter", "Perplexity"],
      "Add": [/* DuckDuckGo Lite, Searx, MetaGer, Startpage, Mojeek */]
    }
  }
}
```

## Files Changed

| File | Action | Description |
|---|---|---|
| `home/common/librewolf.nix` | **Create** | Nix module: extension list, prefs, generates policies.json + overrides.cfg |
| `home/common/default.nix` | **Modify** | Import `librewolf.nix`, remove inline `.librewolf/librewolf.overrides.cfg` source |
| `scripts/install-librewolf.sh` | **Modify** | Add policies.json deployment after app copy |
| `config/librewolf/librewolf.overrides.cfg` | **Delete** | Replaced by Nix-generated version |

## Install Flow

1. `just switch` triggers nix-darwin activation
2. Home-manager generates `~/.librewolf/policies.json` and `~/.librewolf/librewolf.overrides.cfg`
3. `install-librewolf.sh` downloads/verifies/installs LibreWolf DMG
4. Script copies `~/.librewolf/policies.json` → `/Applications/LibreWolf.app/Contents/Resources/distribution/policies.json`
5. On next LibreWolf launch: extensions auto-install from AMO, prefs applied from overrides.cfg

## What Stays Manual

- MetaMask and polkadot{.js} extension installation
- Extension configuration (uBlock filter lists, SponsorBlock categories, etc.)
- Bookmarks
- Login sessions (Bitwarden vault unlock, etc.)
